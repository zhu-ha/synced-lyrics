### Usage

1.  Run the script:
    ```bash
    python3 lyrsync.py
    ```
2.  When prompted, **drag and drop** the audio file path.
3.  Next, **drag and drop** the **.lrc lyrics file path**.
4.  Specify the number of times to play (e.g., `1` for single, `0` for loop).

 ### Requirements

* Python 3.7 or later.
* One supported audio player (e.g., mpv, ffplay) must be installed.
